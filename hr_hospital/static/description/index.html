HR Hospital Module

The HR Hospital module is designed to manage hospital operations within the Odoo environment. This module includes functionality for managing doctors, patients, diseases, diagnoses, and visits.

Key features include:

  • Separate models for Doctors, Patients, Diseases, and Visits.
  • Custom user groups and permissions.
  • Easy tracking of patient visits, diagnoses, and doctor-patient assignments.
  • Wizards for bulk updates and report generation.
  • Hierarchical disease structure with demo data.
  • Detailed reporting capabilities for doctor performance and patient diagnoses.
  • Calendar, kanban, and pivot views for efficient management.
